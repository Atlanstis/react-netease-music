## 配置

### vscode 配置 import @ 路径提示及代码智能提示

> 1、安装插件：Path Intellisense
> 2、配置：
>
> ```
>     "path-intellisense.mappings": {
>         "@": "${workspaceRoot}/src"
>     }
> ```
